🧱 MARTES — Título y Alcance (paso a paso real)
1️⃣ QUÉ ES (explicado simple)

Imagina que vas a construir una casa.

El título es el nombre del proyecto (qué casa vas a hacer)
El alcance es decir:
👉 qué habitaciones sí vas a construir
👉 cuáles quedan para después

Si esto queda claro, tu sistema crece ordenado.

2️⃣ CÓMO FUNCIONA (lo haces en 4 bloques)

Hoy vas a crear un solo documento con 4 partes:

🔹 Paso 1 — Elegir el título base

Tienes varias ideas, ahora eliges una versión clara y defendible.

👉 Te propongo este (alineado a todo lo que has dicho):

Título tentativo:

“Plataforma web para la enseñanza de fundamentos y patrones de diseño mediante lecciones interactivas orientadas a la toma de decisiones en desarrollo de software”

dice qué construyes → plataforma web
dice qué enseña → fundamentos + patrones
dice cómo enseña → lecciones interactivas
dice para qué → tomar decisiones
🔹 Paso 2 — Definir el ALCANCE (el corazón del ladrillo)
